## 🔄 Data Flow (Behind the Scenes)

```
Demo mode activated
    ↓
Yahoo Finance API (live prices)
    ↓
DataAgent updates MarketState (every 2 seconds)
    ↓
AnalystAgent calculates indicators (every 5 seconds)
    ↓
SignalAgent queries Claude for trade ideas (every 10 seconds)
    ↓
RiskAgent validates position sizing
    ↓
SSE broadcast to frontend (live streaming)
    ↓
Dashboard displays in real-time
```

## 📱 If Demo Mode Doesn't Start

**Common issue:** Old orchestrator still running on port 9010

**Quick fix:**
```bash
# Kill old processes
taskkill /F /IM python.exe

# Restart orchestrator
cd F:\Algok\realswing
python orchestrator.py

# Then try demo mode again
python demo_mode.py start
```
